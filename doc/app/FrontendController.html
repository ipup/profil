CHECKCODE: POST ‘checkcode’ Params:

  • pid : user’s id

  • pcode : code posted by the visitor

Our privacy function. Will compare visitor’s entered code with user’s defined code. Will create session and increment code’s use counter if code is valid Will display an error message if code is invalid Will respond with redirection to page where user came from 

     # File app/controllers/frontend_controller.rb, line 265
265:   def checkcode

266:     # Try to find if our user has create the code entered by our visitor.

267:     code = InviteCode.where(:user_id => params[:pid],:code => params[:pcode]).first

268:     

269:     if !code.nil? # Code valid

270:       # Increment use counter and saving object

271:       code.use_counter += 1

272:       code.save

273:       

274:       # Create the session with our user id in it so we know what pages he/she can access.

275:       # By doing that, we make sure he/she won't access another user's private informations.

276:       session[:code] =  params[:pid]

277:       redirect_to(request.referer, :notice => 'Log in successful')

278:     else # Code invalid

279:       redirect_to(request.referer, :notice => 'Code not valid.')

280:     end

281:   end
